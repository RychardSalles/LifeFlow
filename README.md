# 🌊 LifeFlow

Sistema de organização pessoal desenvolvido como projeto acadêmico, com foco em gerenciamento de tarefas, eventos e acompanhamento de produtividade.

## 📋 Sobre o Projeto

O LifeFlow foi criado para ajudar usuários a organizar suas atividades diárias de forma simples e intuitiva.

A aplicação permite:

- Cadastro de usuários
- Login com autenticação
- Gerenciamento de tarefas
- Gerenciamento de eventos
- Dashboard com informações de produtividade
- Visualização de tarefas por data
- Acompanhamento de tarefas concluídas
